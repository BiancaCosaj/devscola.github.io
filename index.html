¿Quieres aprender a programar en Valencia?

Devscola es una escuela de programación situada en Valencia,
Avd Catalunya nº11, Entresuelo (GEEKSHUBS)

Devscola es una escuela que enseña a programar aplicaciones web. La enseñanza es puramente práctica de manera que a el alumno adquiere competencias profesionales que puede demostrar con un portafolio de aplicaciones web en Github y Bitbucket. Nuestros alumnos son estudiantes y/o profesionales que quiere iniciar su carrera en la programación. También emprendedores que quieren desarrollar su propia idea. Acogemos gente de España y también extranjeros. Nuestro selecto equipo de mentores guían la formación individualmente garantizando que los objetivos se cumplan semana a semana. =======

Al finalizar el periodo de aprendizaje en Devscola, entre 3 y 6 meses dependiendo del alumno, saldrás con las competencias necesarias para trabajar en empresas de software nacionales y de ciudades como Londres. >>>>>>> c9c0b3ebd9194353b101a53640a5bebc6fd77138

    ¿Qué voy a aprender?

    • Construye complejas aplicaciones webcon Ruby on Rails, un potente framework para el desarrollo web.

    • Javascript y jQuery te ayudaran a realizar aplicaciones web dinámicas.

    • Publica tus aplicaciones usando Git de forma rápida sencilla y gratuita.

    • Lo fundamental es siempre importante aprende como hacer que la página web luzca con HTML5.

    • Aprende en MySQL los conceptos de tablas, vistas, IDs, entidades relacionales. etc

    • AGILE AND LEAN

      You will learn modern lean startup principles and agile methodologies used in software development.

    ¿A quién va dirigido?

    Si estas acabando tus estudios y quieres hacer el PFC

    O eres un emprendedor que quiere realizar una web app

    Para aquellos que han empezado a programar recientemente y quieren formarse para encontrar trabajo
